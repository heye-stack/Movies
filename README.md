一个现代化的Android电影应用，采用MVVM架构，使用Jetpack Compose构建响应式UI，Firebase提供后端支持，Cloudinary处理媒体资源。


## 🚀 技术栈


- **100% Kotlin** - 完全使用Kotlin开发
- **Jetpack Compose** - 声明式UI框架
- **Firebase** 套件:
- Firestore - 实时数据库
- Storage - 文件存储
- **Cloudinary** - 云端媒体管理
- exoplayer 安卓原生播放器
- **协程 & Flow** - 异步处理






### 亮点功能
- 响应式UI - 适配各种屏幕尺寸
- 实时数据同步 - 通过Firestore实现
- 无缝媒体上传 - 集成Cloudinary SDK
- 动画过渡效果 - 基于Compose动画API
